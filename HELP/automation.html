Automation

Selecting the Automate button (see Controls) will put the system into automatic rotation.

This means that the system will continually select items at random from the database and add them to the playlist when it becomes empty.

You can still add to and edit the playlist and all other controls work as normal in automatic rotation.

To quit automatic rotation, select the Stop button. As this stops the players in their current state, it is suggested you select Break After then Stop
