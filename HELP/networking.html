AllDay DJ

Networking Screen

AllDay DJ can be setup to use a shared database through the Networking screen.

To use the local database (the database in the AllDay DJ folder) select Use local Database & Settings. Otherwise select Use Shared Database & Settings and browse to the required folder.

The Logs and Music Database will be shared. The other settings such as news timings and admin passwords will remain local.
