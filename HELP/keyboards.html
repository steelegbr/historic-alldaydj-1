AllDay DJ

Specialized Keyboards & Input Devices

AllDay DJ will work with some input devices that connect as a keyboard (e.g. DRS 2006 Remote Control Panel, RITA1) or even just a numpad with sticky labels

When programming the keyboard, please use the following table.

FunctionMain KeyboardNumpad
Automate<ALT> + A0
Play<ALT> + P+
Stop<ALT> + S-
Next Item<ALT> + N.
Break After<ALT> + B*
Play Instant Players (1-9)F(1-9)(1-9)
