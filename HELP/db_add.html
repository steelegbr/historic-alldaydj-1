Add To Database

To add an item to the database follow the procedure below.

  1. Click on browse and select the item you wish to add to the database.
  2. Enter the track and artist titles into the correct boxes.
  3. It is also required that you enter the record company and composer in the other boxes. (Use N/A or similar if unknown).
  4. Select the item type from the dropdown menu.
  5. Click on play and skip to the beginning of the track (usually the very start of the file) and select Start.
  6. At the end of the intro, click on Intro to set the intro countdown point.
  7. Skip to the end of the track and select End where you would want the next track to strart.
  8. If the song fades out at the end, select Fades?
  9. click on Add.
